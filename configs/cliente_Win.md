## Creación del Usuario "bchecker" como Administrador en Windows

En el equipo Windows del proyecto se ha creado el usuario `bchecker` como cuenta local administradora utilizando la interfaz gráfica de Windows. El procedimiento aplicado fue el siguiente:

1. Acceso al menú de Configuración de Windows.
2. Selección de "Cuentas" y, dentro de este apartado, "Familia y otros usuarios".
3. Utilización de la opción "Agregar otra persona a este equipo" para crear un nuevo usuario local llamado `bchecker`.
4. Durante el proceso, se definió la contraseña `bchecker121`.
5. Tras la creación, se accedió a la gestión de cuentas y se cambió el tipo de cuenta de `bchecker` a "Administrador", otorgándole privilegios plenos en el sistema.

Esta configuración asegura que el usuario estándar del proyecto tenga capacidad de administración y acceso a todas las funciones del equipo Windows, alineando la gestión de credenciales con el resto del entorno y facilitando operaciones administrativas y de mantenimiento de forma centralizada.
