# Investigación de Tecnologías Utilizadas en el Proyecto

En el proyecto se han utilizado varias tecnologías y herramientas clave para la infraestructura y servicios. A continuación se detallan las principales, la empresa que las desarrolla o gestiona, y ejemplos de empresas que las utilizan en producción.

---

## 1. Ubuntu Server

- **Empresa:** Canonical Ltd.
- **Empresas que lo usan:** Netflix, Wikipedia, Snapchat, eBay.
- **Descripción:** Sistema operativo Linux orientado a servidores, conocido por su facilidad de uso, soporte y comunidad activa.

## 2. MySQL

- **Empresa:** Oracle Corporation.
- **Empresas que lo usan:** Facebook, Twitter, YouTube, Booking.com.
- **Descripción:** Sistema de gestión de bases de datos relacional, ampliamente utilizado en aplicaciones web y empresariales.

## 3. Netplan

- **Empresa:** Canonical Ltd.
- **Empresas que lo usan:** Utilizado principalmente en servidores Ubuntu en empresas como IBM y Dell.
- **Descripción:** Herramienta moderna para la configuración de redes en sistemas Ubuntu.

## 4. Git

- **Empresa:** Software libre, gestionado por la comunidad; GitHub es propiedad de Microsoft.
- **Empresas que lo usan:** Microsoft, Google, Amazon, IBM, muchas startups y proyectos open source.
- **Descripción:** Sistema de control de versiones distribuido, esencial para la colaboración y gestión de código fuente.

## 5. Apache

- **Empresa:** Apache Software Foundation.
- **Empresas que lo usan:** Adobe, IBM, LinkedIn, muchas empresas de hosting.
- **Descripción:** Servidor web robusto y flexible, líder en el mercado desde hace años.

---

## Comparación y Justificación de las Elecciones

### ¿Por qué Ubuntu Server en vez de Debian?

- **Facilidad de uso:** Ubuntu Server ofrece una instalación más sencilla y una mayor cantidad de documentación y soporte para usuarios nuevos.
- **Actualizaciones y soporte:** Canonical proporciona actualizaciones regulares y soporte LTS (Long Term Support), lo que garantiza estabilidad y seguridad.
- **Compatibilidad:** Muchas herramientas modernas (como Netplan) están optimizadas para Ubuntu.
- **Comunidad:** La comunidad de Ubuntu es más grande y activa, lo que facilita la resolución de problemas.

### ¿Por qué MySQL en vez de PostgreSQL?

- **Popularidad:** MySQL es más conocido y utilizado en aplicaciones web.
- **Integración:** Se integra fácilmente con la mayoría de los CMS y frameworks populares.
- **Simplicidad:** Es más sencillo de administrar para proyectos pequeños y medianos.

### ¿Por qué Git y GitHub?

- **Colaboración:** Permite trabajar en equipo de forma eficiente y segura.
- **Control de versiones:** Facilita el seguimiento de cambios y la recuperación de versiones anteriores.
- **Integración:** GitHub ofrece integración con otras herramientas y servicios de CI/CD.
